Introduction

My name is Matt Connors and I am a senior attending Bridgewater State University for Computer Science. Within the field itself, my primary focus is on computer networking, cloud architecture, and web development.

The culmination of these skills have lead to the development of several projects, including a mockup video platform, a functional CMS system, and a site screenshotting API, among others. These projects are primarily build on AWS and Cloudflare, and involved deep intigration with core cloud services.

To manage this infastructure, Pulumi, an IaaC SDK, was used in compliment with Github to update the state of the associated cloud services. On deployment, a Github Action was utilized to automatically update the cloud services of that working branch (i.e prod, dev, staging). Furthermore, by maintaining several branches during development, I was able to simulate a production environment whereby updates are first vetted and tested before committing them to general use.
