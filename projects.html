Project Concept


Description

An feasable final project idea is a citation generator, much like Citation Machine by Chegg. Provided a link, in addition to other necessary information, the app will produce citations in the specified format (MLA, APA, etc).

Front-end Implementation

The initial design mockup with be made in Figma. Using this design, we will construct the necessary interfaces for our app using React and Astro. For interactive interfaces, we will create React components. Furthermore, Astro will be used to fuse the entire app together, as well as for any content-centric components.

  • Figma for initial design
  • Astro, written in HTML, CSS, and Javascript, for static components
  • React for interactive components

Back-end Implementation

Using Terraform, the appropriate cloud infrastructure will be deployed on Cloudflare, which will host our app. The reason we chose Cloudflare was due to its predictable, low-cost pricing model which doesn't charge for bandwidth.

  • Terraform for managing cloud infrastructure
  • Cloudflare as our primary cloud service provider
    • Cloudflare CDN and DDoS protection
    • Cloudflare Workers
    • Cloudflare Pages
  • A Cloudflare Worker to scrape necessary information from webpages, such as author, published date, etc.
  • Potentially, Imgix for serving images
  • Astro (similar to next.js) for page layout, and static components.
